## Recommendations

1. **Remove all non-existent tokens** from desktop theme generator
2. **Update token mappings** to use correct OpenCode token names:
   - Use `surface-*-base` instead of `*-base`
   - Use `surface-*-hover` instead of `*-hover`
   - Use `text-*` instead of `foreground*`
   - Remove all `*-active` tokens except those that actually exist
   - Remove all `*-text` tokens except those that actually exist

3. **Keep only existing tokens** that are actually generated by resolve.ts and used in the OpenCode app.
